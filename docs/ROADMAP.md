# Development Roadmap

> **Note**: This document will be expanded in the future with detailed timeline and milestones.

## Completed

- PhysioNet dataset integration
- Pan-Tompkins R-peak detection
- Real-time streaming visualization
- Multi-method validation framework
- AD8232 data collection pipeline
- Automated data processing pipeline (master controller files)
- Automated firmware flashing
- Raw data packet integrity analysis

## In Progress

- Algorithm robustness testing across datasets
- Data packet restructuring 
- BLE/WiFi streaming implementation

## Planned

- Enhancement of R-peak detection algorithm (dynamic threshold, filtering and noise reduction, etc.)
- AD8232 AFE final integration
- Battery power management
- Mobile/web interface
